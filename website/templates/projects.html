CFB Assistant

CFB Assistant demonstrates the integration of ChatGPT with a knowledge base to create a subject matter expert chatbot. CFB Assistant answers questions about the 2022 college football season. It uses semantic search to find articles that are most likely to contain the answers to your question. It then places that article into the context of a chatGPT API call and returns the response. This same approach can be used to create subject matter expert bots for any topic.

Deli Digest

Deli Digest is a blog site that uses autogenerated content to create posts about my fantasy football league. It combines information from the Sleeper League API with ESPN game recaps and sends that information to a LLM to generate stories for the league. Deli Digest has functionality to process and generate the posts in the admin panel of the site.
